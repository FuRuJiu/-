中性代词

中性代词的特点

  1. 无性数变化
  2. 作直接宾语，代替一个不定式动词所表示的行为。

    3. 例如：Partez, puisque vous le voulez.（既然你想走，那就走吧。）

  3. 做表语，代替一个表示身份或者职业的名词。

    4. 例如：

    • Est-ce que vous êtes journaliste?（你是记者吗？）
    • Oui, je le suis.（是的，我是。）
  4. 代替一个形容词。

    5. 例如：

    • Êtes-vous fatigué?（你累了吗？）
    • Oui, je le suis.（是的，我累了。）
  5. 代替一个句子（所包含的内容）。

    6. 例如：

    • Vous êtes satisfait?（你满意吗？）
    • Je le vois.（我看得出来。）
